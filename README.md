# 科研合作项目管理系统
大二写的一个javaweb项目
<br/>
## 一、主要功能
 * 数据可按照给的模板进行导入<br/>
 * 按需求导出所需数据为Excel表<br/>
 * 简单的数据统计与分析<br/>
